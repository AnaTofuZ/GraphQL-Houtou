# Legacy Test Snapshot

このディレクトリは、greenfield 再設計前のテスト資産を退避したものです。

- `original-t/`
  - 旧実装で使っていた `t/` ディレクトリのスナップショット

意図:

- 互換確認に使えるケースを後から抜き出せるようにする
- 新しい runtime / VM の開発時に、既存仕様との差分を比較できるようにする
- 旧実装を大胆に削除しても、テストケース自体は参照可能なままにする

注意:

- ここにあるテストは、そのまま新実装へ適用できるとは限りません
- 新 runtime 向けには別のテスト harness / fixture / benchmark を再構築する想定です
